Clustering Spotify Users by Genre to Enhance Personalization

This project focused on applying unsupervised machine learning techniques to segment Spotify users based on their genre preferences. By analyzing real and simulated user listening data, we aimed to identify distinct user clusters to enhance music personalization and targeted marketing strategies.

We utilized advanced data science methods, including clustering algorithms (like K-Means) and dimensionality reduction techniques such as t-SNE, to visualize user segmentation in a meaningful way. A Flask/PostgreSQL backend was implemented to manage the data, providing a robust system for analysis.

The insights gained from this project can be leveraged to deliver more relevant playlist recommendations, improve music discovery features, and inform marketing campaigns by targeting specific user segments with tailored content.
